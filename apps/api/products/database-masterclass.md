# Database Design Masterclass

An in-depth masterclass covering everything from basic SQL to advanced distributed database concepts.

## Course Outline

### Section 1: SQL Fundamentals (2 hours)

- ACID properties explained
- Normal forms (1NF, 2NF, 3NF, BCNF)
- Relationships and constraints
- Transactions and concurrency control

### Section 2: Indexing & Performance (3 hours)

- B-tree and hash indexes

- Index anti-patterns

### Section 3: Advanced Topics (4 hours)

- Sharding strategies
- Replication patterns
- Data warehousing
- Time-series databases
- NoSQL trade-offs

### Section 4: Real-World Design (3 hours)

- Designing for scale
- Backup and disaster recovery
- Monitoring and alerting
- Performance tuning case studies
